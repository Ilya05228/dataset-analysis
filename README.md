# АНАЛИЗ ДАННЫХ КРИПТОВАЛЮТ

## Описание проекта

Проект для анализа данных криптовалют с использованием Python. Включает анализ объемов торгов, цен, волатильности, корреляций и прогнозирование методом линейной регрессии.

## Структура проекта

- `main.py` - основной скрипт анализа
- `files/dataset/` - папка с CSV файлами криптовалют (49 файлов)
- `files/output/` - папка с результатами анализа
- `requirements.txt` - зависимости Python
- `LICENSE` - лицензия проекта

## Установка и запуск

```bash
# Создание виртуального окружения
python3 -m venv .venv

# Активация виртуального окружения
source .venv/bin/activate

# Установка зависимостей
pip install -r requirements.txt

# Запуск анализа
python main.py
```

## Особенности анализа

1. **Анализ объемов:** Расчет годовых объемов торгов и выявление топ-10 криптовалют
2. **Анализ цен:** Визуализация фактических и нормализованных цен
3. **Волатильность:** Расчет дневной волатильности и кластеризация
4. **Корреляционный анализ:** Построение тепловых карт корреляций
5. **Прогнозирование:** Линейная регрессия для прогноза цен на 30 дней
6. **Анализ рисков:** Расчет максимальных просадок и доходности

## Математические методы

- Линейная регрессия: $y = \beta_0 + \beta_1 x + \epsilon$
- t-тест Стьюдента: $t = \frac{\bar{X}_1 - \bar{X}_2}{\sqrt{\frac{s_1^2}{n_1} + \frac{s_2^2}{n_2}}}$
- Корреляция Пирсона: $\rho(X,Y) = \frac{\text{cov}(X,Y)}{\sigma_X \sigma_Y}$
- Функция потерь MSE: $J = \frac{1}{n}\sum(y_i - \hat{y}_i)^2$
