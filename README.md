# Анализ данных криптовалют

## Цель работы

Провести комплексный анализ данных криптовалют для выявления закономерностей в поведении цен, объемах торгов, волатильности и корреляциях между различными цифровыми активами. Разработать подход к прогнозированию цен с использованием методов машинного обучения.

**Что хотим добиться:** Понимание того, как ведут себя цены разных криптовалют относительно друг друга.

**Что хотим добиться:** Определить, какие криптовалюты наиболее рискованные для инвестиций.

### 6. Прогнозирование методом линейной регрессии

**Цель:** Прогнозирование цен криптовалют на 30 дней вперед.

**Формулы:**
- Линейная регрессия: $$y = \beta_0 + \beta_1 x + \epsilon$$
- Функция потерь MSE: $$J(\beta_0, \beta_1) = \frac{1}{n} \sum_{i=1}^{n} (y_i - \hat{y}_i)^2$$
- MAE: $$\frac{1}{n} \sum |y_i - \hat{y}_i|$$
- MAPE: $$\frac{100\%}{n} \sum \left| \frac{y_i - \hat{y}_i}{y_i} \right|$$

**Что хотим добиться:** Создать модель для краткосрочного прогнозирования цен криптовалют.

**Анализ и результаты:**

**Таблица 5. Результаты линейной регрессии (прогноз на 30 дней)**

| Криптовалюта | MAE (USD) | MAPE (%) | R² |
|--------------|-----------|----------|----|
| Tether | $0.00 | 0.0% | -0.034 |
| Bitcoin | $11682.88 | 13.1% | -38.399 |
| Ethereum | $673.35 | 22.1% | -33.108 |
| Usd Coin | $0.00 | 0.0% | 0.126 |
| Solana | $29.40 | 22.9% | -37.655 |

![Регрессия Bitcoin](files/output/regression_Bitcoin.png)

![Регрессия Ethereum](files/output/regression_Ethereum.png)

![Регрессия Solana](files/output/regression_Solana.png)

**Вывод:** Линейная регрессия показывает ограниченную эффективность для прогнозирования цен криптовалют из-за их высокой волатильности и нелинейного поведения. Отрицательные значения R² указывают на то, что модель хуже, чем простое среднее значение.

### 7. Анализ рисков и доходности

**Цель:** Оценка потенциальных рисков и доходности инвестиций в разные криптовалюты.

**Формулы:**
- Максимальная просадка: $$\text{Drawdown} = \frac{P_{\min} - P_{\max}}{P_{\max}} \times 100\%$$
- Доходность: $$\text{Return} = \left( \frac{P_{\text{конец}}}{P_{\text{начало}}} - 1 \right) \times 100\%$$

**Что хотим добиться:** Понимание соотношения риск/доходность для разных криптовалют.

## Общий вывод

1. **Концентрация рынка:** Наблюдается значительная концентрация объемов торгов в топ-10 криптовалют, с доминированием Tether и Bitcoin.

2. **Диверсификация рисков:** Криптовалюты демонстрируют широкий диапазон волатильности - от 0.16% для стабильных монет до 7.88% для высоковолатильных активов.

3. **Корреляции:** Обнаружены сильные корреляции между ценами и волатильностью различных криптовалют, особенно среди альткоинов, что важно учитывать при диверсификации портфеля.

4. **Прогнозирование:** Линейная регрессия показывает ограниченную эффективность для прогнозирования цен криптовалют из-за их нелинейного поведения и высокой волатильности.

5. **Риск/доходность:** Высоковолатильные криптовалюты предлагают высокий потенциал доходности, но сопряжены с существенными рисками. Стабильные монеты подходят для сохранения капитала.

6. **Рыночная динамика:** Объемы торгов демонстрируют значительную изменчивость, отражая общую рыночную активность и настроения инвесторов.

## Рекомендации для инвесторов

1. **Консервативным инвесторам:** Использовать стабильные монеты (Tether, Usd Coin) для хеджирования рисков и сохранения капитала.

2. **Агрессивным инвесторам:** Рассмотреть высоковолатильные активы (Solana, Ethereum) для потенциально высокой доходности, но с обязательным управлением рисками и стоп-лоссами.

3. **При диверсификации:** Учитывать корреляции между криптовалютами для эффективного распределения активов и снижения общего риска портфеля.

4. **Для прогнозирования:** Использовать более сложные модели (LSTM, Prophet) вместо линейной регрессии для учета нелинейных паттернов и сезонности.
