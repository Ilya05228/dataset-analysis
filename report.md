## Основная проблема анализа криптовалют

Сегодня каждая криптовалюта, будь то стабильная монета или высоковолатильный актив, ориентирована на привлечение инвесторов через разные модели доходности. Для стабильных монет ключевым является низкая волатильность, обеспечивающая сохранение капитала, в то время как высоковолатильные активы имеют потенциал высокой доходности. Разрабатываются комплексные стратегии для анализа данных, чтобы понять поведение цен по времени и корреляции между активами [2]. Данные стратегии принимаются для предупреждения ошибок в инвестиционном планировании и оптимизации доходов. Часто проблемы возникают в тех местах, где стабильные монеты имеют низкую волатильность, но низкую доходность, а высоковолатильные активы — наоборот, высокую потенциальную доходность, но высокий риск. В данном случае анализ больших данных позволяет выявить такие паттерны, как преобладание корреляции между определенными парами криптовалют. Нарушения в анализе данных приводят к плачевным последствиям, таким как неверные прогнозы роста [9]. Поэтому очень важно не допускать ошибок в обработке больших данных на рынке криптовалют [10].

## Математическая постановка задачи

В настоящее время большинство методов анализа больших данных используют статистику и машинное обучение без сложных функций активации [15]. Проведем математический анализ сравнения групп без использования функций активации:

Рассмотрим для примера две группы: стабильные монеты (S) и высоковолатильные активы (V).

Для сравнения средних значений используется t-тест Стьюдента:

$$t = \frac{\bar{X}_S - \bar{X}_V}{\sqrt{\frac{s_S^2}{n_S} + \frac{s_V^2}{n_V}}}$$
где $\bar{X}$ — среднее, $s^2$ — дисперсия, $n$ — размер выборки.

Для прогнозирования используется линейная регрессия:
$$y = \beta_0 + \beta_1 x + \epsilon$$
где $y$ — целевая переменная (цена), $x$ — время, $\beta$ — коэффициенты.

Вектор параметров будем искать так же, как в задачах по оптимизации:
$$\theta = \arg\min J(\theta)$$
где $J(\theta) = \frac{1}{2n}\sum(h(x_i) - y_i)^2$.

Так же для примера подберем множество пар с тестовыми данными:
$$X = [2020, 2021, 2022, 2023, 2024]$$$$y = [18.8, 21.9, 24.9, 27.5, 30.2] \text{ (средние цены по годам)}$$
В результате расчетов и преобразований мы выяснили, что без сложных моделей можно добиться точного сравнения и прогноза, с MSE < 1.5 для регрессии [4].

## Математическое обеспечение использованное при анализе

Используется линейная регрессия. Для вычисления коэффициентов:
$$\beta_1 = \frac{\sum(x_i - \bar{x})(y_i - \bar{y})}{\sum(x_i - \bar{x})^2}$$$$\beta_0 = \bar{y} - \beta_1\bar{x}$$
Функция потерь MSE:
$$J = \frac{1}{n}\sum(y_i - (\beta_0 + \beta_1 x_i))^2$$
В качестве оптимизатора — градиентный спуск.

**Рис. 12 Симуляция градиентного спуска**

Градиентный спуск:
$$\theta := \theta - \alpha \nabla J(\theta)$$
